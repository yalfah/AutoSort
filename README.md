This is a simple test repo project to walk through some ML training concept, the goal is to create an auto sorting trash bin that can auto sort items placed in its "sorting zone" into trash or recycle and trigger it to be "flushed" down into the correct bin.

A Transfer Learning application designed to classify waste as **Trash** or **Recycle** on edge devices without internet connectivity.

## Prerequisites
- Python 3.8+
- A dataset of images (See Setup)
